Неизменяемые типы данных (immutable types) в Python — это типы данных, значения которых нельзя изменить после создания.

Примеры неизменяемых типов:
str — строка
Пример: user_name = "Artur"
После создания строки нельзя изменить отдельные символы:
user_name[0] = "a"  # Ошибка

int — целое число
Пример: user_id = 1001
Любое изменение числа создаёт новый объект:
user_id = user_id + 5  # Создается новый объект 1006

tuple — кортеж
Пример: rgb_color = (255, 0, 128)
Элементы кортежа нельзя изменять:
rgb_color[0] = 258  # Ошибка

Неизменяемые типы полезны для:
- Использования в качестве ключей словаря.
- Защиты данных от случайного изменения.
